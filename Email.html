How to Provide Feedback on W3C's CSS Validation Service

If your document doesn't validate, and you don't know why, or you have a question about the CSS validator.

First, check the W3C CSS Validator Frequently Asked Questions (FAQ) to see if your question has been answered there.

Note: due to a change in the Java platform, always add a trailing slash when trying to validate the main page of your web site. Use http:/www.w3.org/ instead of http://www.w3.org.

You have several CSS related mailing lists, so please pick the right one before asking a question:

www-validator-css@w3.org

This mailing list is for comments or bug reports on the CSS validator. This is not the right place to ask questions regarding CSS itself.

This mailing list is publicly archived.

To post messages to www-style@w3.org, you have to be subscribed. Messages from non-subscribers are diverted to a human moderator and may (or may not) appear with a delay.

www-style@w3.org

This mailing list is for technical discussion on Web Style Sheets. This is a focused discussion area: all posts to this list must be about style sheets. The list is also the preferred place for discussing working drafts that are published by W3C's working group on CSS (Cascading Style Sheets).

This mailing list is publicly archived.

This list is not for posts of the form "How do I...". For those, please use the Usenet newsgroup comp.infosystems.www.authoring.stylesheets, or mailing lists like hwg-style (for members of the HTML Writers Guild). Website Tips has links to more fora.

To post messages to www-style@w3.org, you have to be subscribed. Messages from non-subscribers are diverted to a human moderator and may appear with a delay.
